Description

At Hekanize, I worked as Lead Frontend Developer on a modular and scalable web application built with Vue.js 3, TypeScript, Tailwind CSS and the Lit framework. I led the frontend team, assigning tasks, mentoring other developers, reviewing code, and ensuring quality and timely delivery. Most of my work focused on the frontend, but I also contributed to backend and infrastructure tasks using PHP (Laravel), Docker and AWS. Among other contributions, I implemented SAML-based authentication flows, designed reusable widget components and published two internal libraries as private npm packages. I also worked closely with the backend and product teams to ensure seamless integration and a high-quality user experience.
